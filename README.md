# C# (c-sharp)
C# is a modern, object-oriented programming language developed by Microsoft as part of the .NET platform. It is used for building Windows applications, web services, games (via Unity), and enterprise software, offering strong typing, garbage collection, and rich framework support. The C# language is defined by a standardized specification and implemented by the Roslyn compiler, with packages distributed via NuGet and running on the .NET runtime.

## APIs

### C# Language
The C# language itself, maintained by Microsoft with a formal ECMA-334 specification.
**Human URL:** [https://learn.microsoft.com/en-us/dotnet/csharp/](https://learn.microsoft.com/en-us/dotnet/csharp/)

### Roslyn (.NET Compiler Platform)
Open-source .NET compiler platform with C# and Visual Basic compilers and code analysis APIs.
**Human URL:** [https://github.com/dotnet/roslyn](https://github.com/dotnet/roslyn)

### .NET Runtime
The .NET runtime hosts and executes C# programs with cross-platform support.
**Human URL:** [https://github.com/dotnet/runtime](https://github.com/dotnet/runtime)

### NuGet Package Manager
NuGet is the package manager for .NET with a documented HTTP API for package discovery and publishing.
**Human URL:** [https://www.nuget.org](https://www.nuget.org)

### ASP.NET Core
Cross-platform web framework for C# used to build web applications and APIs.
**Human URL:** [https://learn.microsoft.com/en-us/aspnet/core](https://learn.microsoft.com/en-us/aspnet/core)
